# CRL Helper — десктопное приложение

Приложение один раз устанавливается на ПК пользователя и поднимает локальный HTTP-сервер (порт 7777), через который сайт проверки сертификатов ФНС загружает списки отзыва. Запросы к ФНС идут с машины пользователя; ввод команд не требуется — достаточно запустить приложение (или включить автозапуск при первом запуске).

## Для пользователей

1. Скачайте приложение для вашей ОС со страницы загрузок сайта (или из раздела проверки сертификатов, если helper не отвечает).
2. Запустите скачанный файл (например, `CRL-Helper.exe` на Windows).
3. При первом запуске приложение спросит: «Запускать приложение при входе в систему?» — при согласии оно будет стартовать автоматически.
4. Оставьте окно приложения открытым (или сверните) и откройте сайт — проверка отзыва будет работать автоматически.

## Для разработчиков: сборка одного исполняемого файла

Требуется Python 3.9+ (только для сборки).

### Установка PyInstaller

```bash
cd desktop-helper
pip install -r requirements.txt
```

### Сборка

Из папки `desktop-helper/`:

```bash
pyinstaller --onefile --console --name CRL-Helper main.py
```

Артефакт появится в `dist/CRL-Helper` (macOS/Linux) или `dist/CRL-Helper.exe` (Windows). Имя без пробелов для совместимости.

Для Windows сборка выполняется на Windows (или в CI). Для macOS — на macOS. Для Linux — на Linux.

### Сборка через npm (из корня проекта)

```bash
# macOS (только на Mac)
npm run build:helper:mac

# Windows (только на Windows)
npm run build:helper:windows
```

Результат: `site/files/CRL-Helper-macos.zip` или `site/files/CRL-Helper-windows.exe`. Linux-сборку запускайте отдельно на Linux через `scripts/build-helper-linux.sh`.

### Размещение бинарников

Скопируйте собранные файлы в каталог загрузок сайта (например, `site/downloads/`) или выложите в GitHub Releases и укажите ссылки в конфиге сайта (см. `site/js/config/revocation-sources.js` или конфиг загрузок).

## Переменные окружения

- `PORT` — порт (по умолчанию `7777`);
- `HOST` — хост (по умолчанию `127.0.0.1`);
- `CORS_ORIGIN` — заголовок `Access-Control-Allow-Origin` (по умолчанию `*`).

## Безопасность

Разрешены только хосты ФНС: `pki.tax.gov.ru`, `cdp.tax.gov.ru`, `uc.nalog.ru`. Остальные URL в параметре `url` отклоняются (защита от использования как открытого прокси).
