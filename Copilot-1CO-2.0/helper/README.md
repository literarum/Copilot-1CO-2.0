# Локальный CRL-helper

Небольшой HTTP-сервер, который загружает списки отзыва (CRL) с серверов ФНС и отдаёт их браузеру. Нужен для проверки отзыва сертификатов **на машине пользователя**: запросы к ФНС идут с вашего компьютера (при наличии доступа в сеть РФ или VPN), а не с сервера приложения.

## Требования

- **Node.js** 18 или новее (для глобального `fetch`).

Установка Node.js: [nodejs.org](https://nodejs.org/) или через пакетный менеджер вашей ОС.

## Запуск

Из корня проекта:

```bash
npm run helper:crl
```

Или напрямую:

```bash
node helper/crl-helper.js
```

По умолчанию сервер слушает порт **7777** на всех интерфейсах (`http://0.0.0.0:7777`).

Переменные окружения:

- `PORT` — порт (по умолчанию `7777`);
- `HOST` — хост (по умолчанию `0.0.0.0`);
- `CORS_ORIGIN` — значение заголовка `Access-Control-Allow-Origin` (по умолчанию `*`).

## Использование

1. Запустите helper на том же компьютере, с которого открываете приложение в браузере.
2. Откройте раздел проверки сертификатов ФНС и загрузите сертификат.
3. Приложение само запросит CRL через `http://localhost:7777/helper?url=<CRL_URL>` и отправит данные на бэкенд для проверки.

Если helper не запущен, в интерфейсе появится подсказка: «Локальный helper не отвечает. Для проверки отзыва в любой сети запустите на этом компьютере: npm run helper:crl».

## Безопасность

- Разрешены только хосты ФНС: `pki.tax.gov.ru`, `cdp.tax.gov.ru`, `uc.nalog.ru`. Остальные запросы отклоняются (защита от использования helper как открытого прокси).
